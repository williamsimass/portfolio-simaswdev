# William Simas — Portfolio

Personal portfolio built to showcase projects, skills, and professional experience as a Full Stack Developer.

Live at: **[williamsimas.dev](https://williamsimas.dev)** *(or your deploy URL)*

---

## Tech Stack

| Layer | Technology |
|---|---|
| Framework | React 19 + Vite 7 |
| Styling | Tailwind CSS v4 |
| Animations | Framer Motion |
| Carousel | Embla Carousel |
| Icons | Lucide React |
| Language | JavaScript (JSX) |

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
